![](/assets/img/VSC/Install.png){: width="800" .normal }

<br>

2. 다운받은 exe 파일 실행
<br>

1. Git을 사용하고자 할 시 체크 란에서 선택
    - [x] Code을(를) 지원되는 파일 형식에 대한 편집기로 등록합니다.
    - [x] PATH에 추가(다시 시작한 후 사용 가능)
  
    실행 후 두 칸을 체크 (기본 체크 되어있음)
