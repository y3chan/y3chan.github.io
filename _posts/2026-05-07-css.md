# CSS란?
CSS(Cascading Style Sheets)는 HTML이나 XML로 작성된 문서의 표시 방법을 정의하는 스타일 시트 언어이다.

HTML이 문서의 구조를 표시한다면 CSS는 HTML 요소들이 어떤 디자인과 배치로 표시될지를 정의한다.

결과적으로 웹 페이지를 꾸미기 위해 작성하는 코드이고, HTML 문서의 요소들에 원하는 스타일을 적용할 수 있다.
<br>

## 기초
HTML에 대해 약간의 이해가 생겼다면 CSS가 어떤 식으로 HTML의 구조를 꾸민다는 것인지 알아보자.

예를 들어, 어떤 HTML 문서에 `<p>안녕하세요</p>` 이런 요소가 있다고 했을 때, 다음과 같이 CSS를 작성하면 어떻게 될까?

```
p {
    color: blue;
}
```

이는 모든 `<p>` 요소의 텍스트 색상을 파란색으로 지정하겠다는 의미이고 `안녕하세요` 라는 텍스트의 색이 파란색으로 변할 것이다.
<br>

## ruleset

![](/assets/img/css/ruleset.png){: width="400" .normal }

위와 같은 CSS의 구조를 `ruleset`이라고 한다. `rule` 또는 `CSS rule`이라고도 한다.

- 선택자(selector)</n>
  HTML의 태그와 같은 곳의 이름을 가리킨다. 여기에서는 `p` 태그를 의미한다.
  그 외에도 클래스 선택자, 아이디 선택자 등이 있다.
<br>

- 속성(property)</n>
  어떠한 방식으로 꾸밀 수 있는 방법이다. color는 `<p>` 요소의 속성이고 색을 바꿀 수 있게 해준다. 속성과 속성 값을 구분해주기 위해 속성을 적은 후 `콜론 (:)`을 붙여준다.
<br>

- 속성 값(property value)</n>
  주어진 속성에 있는 수많은 값들 중 하나를 골라 적는다. 즉, color라는 속성의 값에는 `red, blue, yellow ...` 등 여러 색의 정보를 가지고 있다. 그 다음 선언과 구분하기 위해 속성 값 다음에는 `세미콜론 (;)`을 붙여준다.
<br>

- 선언(declaration)</n>
  `color: blue;` 같이 속성과 값을 함께 작성한 한 줄을 선언이라고 한다.
<br>
