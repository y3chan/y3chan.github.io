# HTML에 대하여
## Hyper Text
HTML은 하이퍼텍스트 마크업 랭귀지(Hyper Text Markup Language)의 줄임말이다.

그 중에서 하이퍼텍스트는 문서 안에서 다른 문서나 특정 위치로 연결할 수 있는 텍스트 구조를 의미한다.

예를 들어 지금 `HTML`이라는 문서를 보고 있고, 나는 지금 `HTML 태그`에 대한 문서로 넘어가고 싶을 때 `HTML 태그` 라는 텍스트나 이미지가 클릭한다면 그 문서로 바로 넘어가게 되는 것이다.

일반적으로 순차적으로 정보를 찾는 것에 비해 내가 알고 싶은 것만 골라서 볼 수 있으니 효율적으로 정보를 얻을 수 있을 것이다.

그럼, 하이퍼텍스트가 다른 문서로 이동하게 해주는 것이라고 이해했을 때 마크업 랭귀지는 뭘까?
<br>

## Markup Language
마크업은 `태그(tag)`를 사용해 어떠한 컨텐츠, 문서 구조를 표시하는 형식을 말하는 것이고 이러한 마크업의 언어 즉, 이런 형식을 따르는 하나의 언어라고 생각하면 된다.

결론적으로 HTML은 하이퍼텍스트를 비롯해 태그를 이용하여 일련의 구조화된 문서를 작성하기 위해 생겨난 것이라고 볼 수 있다.
<br>

## Tag
HTML에서 Tag는 뭘까?

HTML에선 `<a></a>` 이런 태그를 사용하게 되는데 `<a>` 이런 식으로 쓰여져 있는 것이 태그이다. 여기서 `<a>`는 하이퍼링크를 만드는 태그이며, `href` 속성에 이동할 주소를 작성한다.

만약 `<a href="https://naver.com">네이버</a>` 이런 식으로 태그를 감싼다면 네이버라는 텍스트가 화면에 출력되고 그 주소로 이동할 수 있는 것이다.
<br>

## Web Browser
그럼 우리가 흔히 쓰는 웹 브라우저 (크롬, 엣지, 파이어폭스 ... ) 등의 프로그램은 기본적으로 무엇을 해주는 프로그램일까?

기본적으로 작성되어진 HTML의 문서를 가져다 그 내용을 해석하여 화면에 표시하는 프로그램이라고 보면 된다.

HTML 문서의 내용을 해석하여 화면에 출력하는 과정을 `렌더링`이라고 한다.
<br>

## HTML 요소(Element) 구조

![](/assets/img/html/element.png){: width="400" .normal }

- 여는 태그(Opening tag)</n>
  태그의 이름을 `<>` 괄호 안에 넣는다. 예를 들어, `<a>` 태그를 사용한 후 내용에 어떤 사이트의 주소를 적어두면 해당 주소로 넘어갈 수 있는 하이퍼링크가 만들어질 것이다. 이러한 태그는 열린 순간부터 효과가 적용되기 시작한다.
<br>

- 닫는 태그(Closing tag)</n>
  태그를 열었는데 닫지 않으면 여는 태그 이후의 내용들이 그 태그에 영향을 받을 수도 있다. 그래서 내가 원하는 내용만 해당 태그를 활용할 수 있도록 `슬래시(/)`를 통해 태그를 닫을 수 있다.

  `</a>` 이런 식으로 태그 이름 앞에 `/`를 붙여 태그를 닫을 수 있고, 태그를 닫지 않으면 브라우저가 의도하지 않은 구조로 해석할 수 있으므로, 대부분의 태그는 올바르게 닫아주는 것이 좋다.
<br>

- 내용(Content)</n>
  텍스트를 적으면 된다. 위와 같이 `<a>` 태그로 감싸져있는 경우에는 주소를 적어주면 되는 식이다.
<br>

- 요소(Element)</n>
  여는 태그, 닫는 태그, 내용을 포함한 전체 구조를 요소라고 한다.
<br>

## 기본 구조
```
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
    </head>
    <body>
        <p>Content</p>
    </body>
</html>
```
<br>

- `<!DOCTYPE html>`
  
  이 문서가 HTML5 형식의 문서라는 것을 브라우저에게 알려주는 선언이다.
<br>

- `<head>`
  
  title, 문자 인코딩, CSS, JavaScript, 검색 엔진 정보 등의 메타데이터를 담는 영역이다. 화면에 직접 보이는 내용은 아니다.
<br>

- `<meta charset="utf-8">`
  
  `meta`: 문서의 부가 정보를 담는 태그이다.</n>
  `charset="utf-8"`: 문자 인코딩 방식을 UTF-8로 설정해준다는 의미다. UTF-8은 한글을 포함한 대부분의 문자를 표현할 수 있는 대표적인 문자 인코딩 방식이다.
<br>

- `<body>`
  
  브라우저 화면에 표시되는 내용이 들어가는 영역의 태그이다.</n>
  우리가 흔히 보는 웹 페이지의 내용들이 이 태그 안에 작성되어 있다고 봐도 된다.
<br>

- `<p>`
  
  문단(paragraph)의 약자로 텍스트를 입력하면 내용 앞 뒤로 빈 줄이 생기며 문단을 적을 수 있는 태그이다.</n>
  줄 바꿈이 없는 경우에도 화면 너비를 넘어가면 자동으로 아래 줄로 넘기며 표시된다.
<br>
